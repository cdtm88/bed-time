    **Step 2: Replace src/app/api/generate/route.ts** with the complete route handler.

    The file must:

    1. Keep `export const runtime = "edge"` at the top

    2. Import and instantiate the Anthropic client:
       ```typescript
       import Anthropic from "@anthropic-ai/sdk";
       import { validateInput } from "@/lib/schemas";
       import { getReadingLevel } from "@/lib/age-levels";
       import { buildSystemPrompt, buildUserMessage, getWordCount, getMaxTokens } from "@/lib/prompts";
       ```
       Instantiate client at module level: `const client = new Anthropic();`
       (SDK reads ANTHROPIC_API_KEY from process.env automatically)

    3. In the POST handler:
       a. Parse request body with `await request.json()` wrapped in try/catch (malformed JSON returns 400)
       b. Call `validateInput(body)` — if returns non-null, return `Response` with status 400, Content-Type `application/json`, body `JSON.stringify({ error: validationError })`
       c. Destructure validated input: `const { name, age, theme, duration } = body as GenerateInput`
       d. Get reading level: `const readingLevel = getReadingLevel(age)`
       e. Get word count: `const targetWords = getWordCount(duration as 5 | 10 | 15)`
       f. Build prompts: `const systemPrompt = buildSystemPrompt(readingLevel, targetWords)` and `const userMessage = buildUserMessage(name, theme)`
       g. Get max tokens: `const maxTokens = getMaxTokens(duration as 5 | 10 | 15)`

    4. Call Claude with streaming (per D-02 model, D-03 plain text):
       ```typescript
       const stream = await client.messages.create({
         model: "claude-sonnet-4-6",
         max_tokens: maxTokens,
         system: systemPrompt,
         messages: [{ role: "user", content: userMessage }],
         stream: true,
       });
       ```

    5. Convert async iterable to ReadableStream (per RESEARCH.md Pattern 1):
       ```typescript
       const encoder = new TextEncoder();
       const readable = new ReadableStream({
         async start(controller) {
           try {
             for await (const event of stream) {
               if (
                 event.type === "content_block_delta" &&
                 event.delta.type === "text_delta"
               ) {
                 controller.enqueue(encoder.encode(event.delta.text));
               }
             }
             controller.close();
           } catch (err) {
             controller.error(err);
           }
         },
       });
       ```

    6. Return response with Content-Type `text/plain; charset=utf-8` (per D-03, NO SSE framing):
       ```typescript
       return new Response(readable, {
         headers: { "Content-Type": "text/plain; charset=utf-8" },
       });
       ```

    7. Wrap the entire Claude call in try/catch. On error, return:
       ```typescript
       return new Response(
         JSON.stringify({ error: "Story generation failed. Please try again." }),
         { status: 500, headers: { "Content-Type": "application/json" } }
       );
       ```
       NEVER expose Claude API error messages, model name, or API key in the error response.
