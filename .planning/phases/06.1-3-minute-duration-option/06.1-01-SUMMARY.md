# Phase 06.1 Plan 01: 3-Minute Duration Option Summary

**One-liner:** Added 3-minute duration (450 words, 900 tokens) with compact narrative arc (1-2/2/1-2 paragraphs) for short bedtime stories.

## What Was Done

### Task 1: Duration 3 config, validation, and UI toggle (TDD)

Added duration 3 as the first option across all layers:

- **src/lib/prompts.ts**: Added `3: { words: 450, maxTokens: 900 }` to DURATION_CONFIG
- **src/lib/schemas.ts**: Updated VALID_DURATIONS to `[3, 5, 10, 15]` and error message to list all four durations
- **src/components/duration-toggle.tsx**: Updated DURATIONS to `[3, 5, 10, 15]` so "3 MIN" appears first in the toggle

**Commits:**
- `586ec12` test(06.1-01): add failing tests for duration 3 config and validation
- `2b2c2d0` feat(06.1-01): add 3-minute duration to config, validation, and UI toggle

### Task 2: Compact narrative arc for short stories (TDD)

Added conditional arc structure in `buildSystemPrompt` based on `targetWords < 500`:

- **Compact arc** (3-minute stories): 1-2 paragraphs beginning, 2 paragraphs middle (wonder only, no tension), 1-2 paragraphs ending
- **Standard arc** (5+ minute stories): unchanged 2-3/3-4/2-3 paragraph structure
- **Ending wind-down**: identical in both paths (sentence taper, heavy eyelids, slow breathing, blanket warmth, yawn, sleep invitation)

**Commits:**
- `a0b022e` test(06.1-01): add failing tests for compact narrative arc
- `03c6a27` feat(06.1-01): add compact narrative arc for short stories
