**Gathered:** 2026-03-26

<domain>
## Phase Boundary

Add a 3-minute reading duration option alongside the existing 5 / 10 / 15 minute options. Scope: DURATION_CONFIG word count + token limit, schema validation, UI toggle, and prompt arc adjustment for short stories. No new UI components — the DurationToggle already exists and just needs a new entry.

</domain>

<decisions>
## Implementation Decisions

### Word count and token limit
- **D-01:** 3-minute word count target: **450 words** (linear projection from 5 min → 750 words)
- **D-02:** maxTokens for 3 min: **900 tokens** (2× word count, consistent with 5/10 min ratio)
- **D-03:** Word count range uses the same ±7% formula as existing durations: `low = Math.round(450 * 0.93)`, `high = Math.round(450 * 1.07)` → 419–482 words

### Narrative arc adjustment
- **D-04:** `buildSystemPrompt()` uses a **compact arc** when `targetWords < 500` (i.e., the 3-minute story). Compact arc paragraph counts:
  - Beginning: 1-2 paragraphs (establish world + character, warm and inviting)
  - Middle: 2 paragraphs (gentle discovery or wonder, no tension)
  - Ending: 1-2 paragraphs (wind-down — sentence taper, sleepy cues, sleep invitation)
- **D-05:** The ending instructions (sentence taper, sleepy sensory cues, sleep invitation) are **unchanged** regardless of story length — those are the most important wind-down elements
- **D-06:** The threshold `targetWords < 500` targets only the 3-min story. Any future duration between 3 and 5 min would also get the compact arc, which is the desired behavior

### UI
- **D-07:** "3 MIN" button appears **first** in the toggle (shortest-first ordering: 3, 5, 10, 15)
- **D-08:** Label is "3 MIN" — consistent with existing button labels ("5 MIN", "10 MIN", "15 MIN")
- **D-09:** `VALID_DURATIONS` updated to `[3, 5, 10, 15]`; error message in `validateInput` updated to reflect the new valid values

### Claude's Discretion
- Exact conditional structure inside `buildSystemPrompt` (ternary, if-else, or extracted helper) — Claude decides what's clearest
