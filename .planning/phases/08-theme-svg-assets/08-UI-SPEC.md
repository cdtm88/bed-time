| Accent shape | #e4e3db (`surface-container-highest`) | A subtle centered shape (circle or rounded square) to indicate "illustration missing" without alarming the user |

## SVG Asset Specification

### Dimensions and Structure

| Property | Value |
|----------|-------|
| Artboard | 200 x 200 px |
| viewBox | `0 0 200 200` |
| Complexity | 3-5 recognizable shapes per theme |
| Style | Clean, iconic storybook — simple vector paths, not complex multi-layer illustrations |
| Stroke width | 2-3 px |
| Stroke linecap | `round` |
| Stroke linejoin | `round` |
| Corner radius | All rectangles use `rx` attribute; no sharp 90-degree corners anywhere |
| File size target | Under 5 KB per SVG (clean paths, no embedded images, no gradients) |

### Prohibited SVG Patterns

- No embedded raster images (`<image>` elements)
- No `<text>` elements (illustrations only, labels are in the HTML)
- No CSS `<style>` blocks inside SVGs (use inline attributes)
- No JavaScript or event handlers
- No external references (`<use href="...">` pointing outside the file)
- No gradients (keep flat fills per DESIGN.md storybook aesthetic)
- No filters or blend modes

## Copywriting Contract

This phase introduces no new pages, CTAs, or user-facing text flows. The only copywriting element is the fallback tile.

| Element | Copy |
|---------|------|
| Primary CTA | Not applicable (no new CTAs in this phase) |
| Empty state heading | Not applicable (theme grid is always populated with all 18 themes) |
| Empty state body | Not applicable |
| Error state (SVG load failure) | No text displayed. Fallback is a silent visual placeholder tile — a tonal background with a subtle shape. The grid layout remains intact and the theme name label still renders below the placeholder. |
| Destructive confirmation | Not applicable (no destructive actions in this phase) |

Rationale for silent fallback: A broken image icon or error text would be alarming in a bedtime story app. The tonal placeholder maintains the calm aesthetic while the theme name label below still communicates which theme the tile represents.

## Interaction Contract

### Fallback Tile Behavior

| Aspect | Specification |
|--------|--------------|
| Trigger | SVG `<Image>` fails to load (network error, 404, corrupt file) |
| Visual result | The 200x200 image area shows a rounded rectangle filled with `surface-container-low` (#f5f4ec) containing a centered circle of `surface-container-highest` (#e4e3db) |
| Interactivity | Tile remains fully clickable and selectable as a theme option |
| Selection state | Same ring treatment as working tiles: `ring-2 ring-primary/30` when selected |
| Transition | 300ms ease-in-out (per DESIGN.md rule: all state changes use this curve) |

### No Other Interaction Changes

All existing theme grid interactions (click to select, radio group keyboard navigation, selected state ring) remain unchanged. This phase does not modify interaction behavior.
