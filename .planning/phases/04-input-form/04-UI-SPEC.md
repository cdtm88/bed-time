## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding within stepper buttons |
| sm | 8px | Compact element spacing, gap between stepper label and buttons |
| md | 16px | Default element spacing, gap between form fields |
| lg | 24px | Section padding, form container horizontal padding |
| xl | 32px | Internal card/section padding (DESIGN.md `spacing-6` = 2rem) |
| 2xl | 48px | Vertical gap between form sections (name/age group vs themes) |
| 3xl | 64px | Page top/bottom padding |

Exceptions: Touch target minimum 44px height for age stepper buttons, duration buttons, and theme tiles (accessibility).

## Component Inventory

### 1. Name Input (Pill)
- Shape: `border-radius: 9999px` (ROUND_FULL)
- Fill: `surface-container-highest` (#e4e3db)
- Height: 48px
- Horizontal padding: 24px
- Text: `body-lg` (Noto Serif, 16px, 400)
- Placeholder: "Your child's name" in `on-surface` at 40% opacity
- Focus state: 2px outer glow of `primary-container` (#9ae1ff), no border change
- Error state: 2px outer glow of destructive (#b3261e at 40% opacity), error text below in destructive color
- Validation: real-time, letters and spaces only, max 30 chars (mirrors `schemas.ts` NAME_REGEX)
- Transition: all state changes 300ms ease-in-out

### 2. Age Stepper
- Layout: horizontal row -- `[ - ]  5  [ + ]`
- Stepper buttons: 44px x 44px circles, `surface-container-highest` fill
- Button icons: minus/plus as text characters "-" and "+" in `on-surface`, `label-md` weight
- Number display: `headline-md` (28px), centered between buttons, min-width 48px
- Range: 0-10, buttons disable at boundaries (opacity 0.3)
- Transition: number change 300ms ease-in-out

### 3. Duration Toggle (Segmented Pill)
- Layout: horizontal row of 3 pill buttons, 8px gap between them
- Each button: `border-radius: 9999px`, height 44px, horizontal padding 20px
- Unselected: `surface-container-low` (#f5f4ec) fill, `on-surface` text
- Selected: gradient fill (`primary` to `primary-container` at 135deg), white (#ffffff) text
- Text: `label-md`, uppercase -- "5 MIN", "10 MIN", "15 MIN"
- Default selection: 10 MIN (middle option)
- Transition: fill change 300ms ease-in-out

### 4. Theme Tile Grid
- Layout: 2-column CSS grid, 12px gap
- Each tile: `border-radius: 1.5rem` (24px), `surface-container-low` fill, padding 16px
- Tile content: SVG illustration (aspect-ratio 1:1, width 100%, `border-radius: 1.5rem`) above theme name (`title-sm`, centered below)
- Selected state: background shifts to `primary-container` at 15% opacity overlay on `surface-container-low`, 2px ring of `primary` at 30% opacity
- Tap target: entire tile is tappable, minimum 44px height guaranteed by content
- SVG source: `<Image src="/themes/{kebab-case}.svg" width={200} height={200} alt="{Theme Name}" />`
- Single-select: tapping a new tile deselects the previous one
- Transition: background/ring 300ms ease-in-out

### 5. Generate Button (Primary CTA)
- Shape: `border-radius: 9999px`
- Fill: linear-gradient 135deg from `primary` (#0c6681) to `primary-container` (#9ae1ff)
- Text: `label-md`, uppercase, white (#ffffff)
- Height: 52px, full-width of form container
- Hover: gradient brightness +10%
- Disabled state (while any required field is empty): opacity 0.4, cursor not-allowed
- Loading state: not needed on button itself (full-screen overlay replaces form)
- Transition: brightness 300ms ease-in-out

### 6. Loading Overlay
- Coverage: fixed position, full viewport (inset: 0), z-index 50
- Background: `surface` (#fbf9f1)
- Content: centered vertically and horizontally
- Headline: `headline-md` -- "Crafting {Name}'s story..."
- Body: `body-lg` -- "This takes about 30 seconds."
- Animation: CSS keyframes, calm and magical (Claude's discretion per D-12 -- stars, gentle glow, moon motif). Use only CSS animations, no external libraries.
- Form is unmounted while overlay is active (D-11)

### 7. Rate Limit Error (Inline)
- Position: below Generate button, 8px margin-top
- Text: `body-lg`, destructive color (#b3261e)
- Form remains visible and interactive

### 8. Validation Error Text
- Position: below the invalid field, 4px margin-top
- Text: `label-md` size (12px), destructive color (#b3261e)
- Appears/disappears with 300ms ease-in-out

## Interaction States

### Form Submission Flow
1. Parent fills all 4 fields (name required, age defaults to 5, duration defaults to 10 min, theme required)
2. Generate Story button becomes active (full opacity) when name is non-empty AND theme is selected
3. On tap: form is unmounted, full-screen loading overlay appears
4. POST to `/api/generate` with `{ name, age, theme, duration }`
5. On success: navigate to `/story` with story text (mechanism at Claude's discretion -- sessionStorage, route state, or URL param)
6. On rate-limit (429): dismiss overlay, re-show form, display inline rate-limit message
7. On other error (500): dismiss overlay, re-show form, display inline generation error message

### Field Defaults
| Field | Default | Reason |
|-------|---------|--------|
| Name | empty | Must be entered by parent |
| Age | 5 | Middle of range, reduces friction |
| Duration | 10 min | Middle option, pre-selected |
| Theme | none | Parent must choose; no sensible default |

### Accessibility
- All form fields have associated `<label>` elements
- Age stepper buttons have `aria-label="Decrease age"` and `aria-label="Increase age"`
- Theme tiles use `role="radio"` within a `role="radiogroup"` with `aria-label="Choose a theme"`
- Duration buttons use `role="radio"` within a `role="radiogroup"` with `aria-label="Story length"`
- Loading overlay uses `role="alert"` and `aria-live="polite"`
- Error messages use `role="alert"`
- Focus is trapped within the loading overlay while it is visible
- Generate button has `aria-disabled` when requirements not met

## Layout

### Mobile-First (320px-768px)
- Form container: max-width 480px, centered, horizontal padding 24px
- Page title: centered, margin-top 64px
- Fields stack vertically with 16px gap between fields, 48px gap before theme grid
- Theme grid: 2 columns
- Generate button: full width, margin-top 32px, margin-bottom 64px

### Tablet+ (769px+)
- Form container: max-width 520px, centered
- Theme grid: remains 2 columns (tiles get slightly larger)
- No layout changes beyond container width -- form is inherently narrow
